Conceptos Clave

¿Qué es el IVA?

El Impuesto al Valor Agregado grava el valor transferido a bienes y servicios. Clic para ver detalles legales.

Base Imponible

Es el monto exacto sobre el cual se calcula el porcentaje de impuesto a pagar. Clic para ampliar definición.

Agente de Retención

Persona o empresa designada por la ley para retener parte del impuesto. Clic para conocer obligaciones.

Tipos de Comprobantes

Comprobantes de Pago

Documentos que acreditan transferencias de bienes o servicios. Clic para ver clasificaciones.

Factura Física

Comprobante de pago tradicional emitido en formato de papel. Clic para ver requisitos de validez.

Factura Electrónica
